# Welcome to @ibecsystems/nuxt3-ui-kit

**@ibecsystems/nuxt3-ui-kit** поможет вам быстро создать приложение Nuxt  используя готовые компоненты.    <br />
Это облегчает создания проектов класса **C и D**.

**@ibecsystems/nuxt3-ui-kit** will help you quickly create Nuxt application using ready-made components.  <br />
This makes it easier to create Class **C and D** projects.
